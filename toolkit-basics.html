Toolkit & Basics

A lookup hub, not a course. The AI tools that carry the workshop and the everyday Workspace moves you need along the way.

A Gem is Gemini after one good briefing. You do not build Gems for the workshop; you use the prepared ones for each step.

AI tools

Tool

Gemini

Drafts from scratch, rewrites, summarizes, structures, and helps you get past the blank page.

Tool

Gems

Prepared assistants for one job at a time: briefing, first contact, value argument, forecast, coaching, and more.

Tool

Deep Research

A Gemini mode that searches many public sources and returns findings with citations. Used by the Lead-Finder and Account-Briefing Gems.

Tool

NotebookLM

Works only from the sources you give it and shows citations, so answers can be traced back.

Tool

Workspace Studio

Turns a recurring trigger-based task into a standing routine, with human approval kept in the flow.

Workspace basics

Gmail

Draft and summarize

Use Help me write and the Gemini side panel, then read and edit before sending.

Meet

Take notes for me

Let Gemini capture notes, summary, and next steps, then correct the Doc before acting.

Docs

Draft, rewrite, summarize

Attach a Doc to a Gem instead of pasting a wall of text, then save the answer as the next Doc in the chain.

Sheets

Ask data questions

Build starter tables, tidy data, ask plain-language questions, and verify before acting.

Slides

Draft an outline

Use Gemini for a starter deck or image idea, then keep structure and polish human.

Vids

Short sales videos

Turn a deck or short brief into a watchable video, especially for quick customer messages.
