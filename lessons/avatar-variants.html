Der Tour-Einstieg

Nüchterne Varianten für die Zielgruppe erfahrener Vertriebsprofis: kein Maskottchen, keine Comic-Optik. Ein glaubwürdiger Mensch oder ein rein funktionales Element.

Jede Zeile ist ein klickloser Nachbau der Tour-Zeile der Startseite. Auch die Sprache ist bewusst sachlicher gefasst („Get the 30-second overview“ statt „I can show you around“); Hüpf- und Blinzel-Animationen entfallen in allen Varianten.

A · Mit Mensch

Die stärkste Ansprache für diese Zielgruppe: eine echte, glaubwürdige Person — jemand wie sie selbst. Keine Illustration.

A1 · Foto eines Kollegen
First time here? Get the 30-second overview.
▶ Start the tour

Platzhalter für das Foto einer echten Person mit Standing in der Zielgruppe: erfahrene:r Vertriebskolleg:in oder Trainer:in, sachliches Porträt, rechteckig mit gelber Kante. Wirkt wie „ein Kollege zeigt mir das“, nicht wie ein Maskottchen. Die Tour-Stimme (Audio) passt dann zur Person. Braucht: ein Foto + Einverständnis.

B · Ohne Mensch, rein funktional

Kärcher ist eine technische Marke: Das Tour-Element als sachliches Werkzeug in Gelb/Anthrazit, im Stil technischer Beschilderung. Keine Gesichter, keine Verspieltheit.

B1 · Piktogramm
First time here? Get the 30-second overview.
▶ Start the tour

Reduziertes Piktogramm im Stil technischer Beschilderung (DIN/ISO-Anmutung): eine Figur zeigt auf die Inhalte. Kein Gesicht, keine Emotion — reine Funktion „hier wird Ihnen etwas gezeigt“. Passt nahtlos zur Marke.

B2 · Rundgang-Route
First time here? Get the 30-second overview.
▶ Start the tour

Die Tour als das, was sie ist: ein kurzer Rundgang über die Stationen der Seite. Sachlich, technisch, ohne jede Figur. Funktioniert auch als wiederkehrendes Symbol für „Orientierung“ im ganzen Auftritt.

B3 · Nur Text, dunkles Band

Orientation

First time here? Get the 30-second overview.

▶ Start the tour

Die nüchternste Lösung: gar kein Bildelement. Ein scharfkantiges Anthrazit-Band mit gelbem Kicker und Button — hebt sich klar von den weißen Kästchen ab und behandelt die Tour wie eine Funktion des Produkts. Die Audio-Stimme bleibt.

C · Blockgestaltung

Unabhängig von der Bild-Entscheidung: der Block scharfkantig (Kärcher-CI „rechteckige Flächen“) und farblich von den weißen, abgerundeten Themen-Kästchen abgesetzt.

C1 · Anthrazit + Piktogramm
First time here? Get the 30-second overview.
▶ Start the tour

Dunkler, scharfkantiger Block mit gelbem Piktogramm-Quadrat und gelbem Button: maximaler Kontrast zu den Kästchen, komplett im Marken-Duktus.

C2 · Weiß mit gelber Kante + Foto
First time here? Get the 30-second overview.
▶ Start the tour

Ruhigere Alternative: weißer, scharfkantiger Block mit kräftiger gelber Kante, kombiniert mit dem Kollegen-Foto. Seriös und persönlich zugleich.

Empfehlung für diese Zielgruppe: A1 (Foto einer glaubwürdigen Person), sobald Foto + Freigabe vorliegen — bis dahin B1 oder B3 mit C1-Blockgestaltung. Dazu durchgängig die sachlichere Formulierung und keine Animationen.
