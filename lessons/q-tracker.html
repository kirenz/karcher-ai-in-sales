Set up a simple account tracker

The lightweight home for your accounts — built by Gemini in one prompt.

You have

Nothing but a blank Google Sheet. That is the point.

You’ll create

An account tracker with the right columns and three example rows showing the format.

Time

About 3 minutes, right inside Sheets.

What you need

A blank sheet — really

A blank spreadsheetOpen a new Google Sheet. No columns to design, no format to invent — Gemini builds the structure.

Do it · in Sheets

Ask for the structure

In your blank sheet, click Ask Gemini (top right) and run the prompt. Gemini often shows a small plan first — a few steps and an Approve button. Just approve it, and the tracker gets built. Then replace the example rows with real accounts, and it is yours.

Copy-paste prompt:

Create a simple account tracker with these columns: account, segment, stage, value, last-contact date, next step, note. Add three example rows so I can see the format.

Open Google Sheets ↗

Check it

Make it yours in two edits

The structure is generic on purpose — now fit it to your world:

  • Rename or add columns to match how you actually work — the tracker serves you, not the template.
  • Replace the example rows with two or three real accounts while the format is fresh.
  • Keep one rule: every account gets a next step. An empty next-step cell is a signal.

Done

✓

Task done

You produced: Your own account tracker — the sheet every other data task builds on.
