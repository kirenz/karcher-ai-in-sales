Clean up messy data

Mixed date formats, inconsistent names, values as text — straightened out, with a change list.

You have

A sheet full of imported mess: dates in three formats, names spelled four ways, values as text.

You’ll create

A cleaned copy on its own new tab, plus a list of every change — your original stays untouched.

Time

About 5 minutes, right inside Sheets.

What you need

The messy table, nothing else

Your messy tableThe imported data open in Google Sheets — or your copy of the Sales-Data practice file.

No messy data at hand?

Use the Sales-Data practice file: deliberately broken formatting — mixed date formats, inconsistent names, values as text. Open it, then File → Make a copy, and clean your copy.

Open the Sales-Data file ↗

Do it · in Sheets

Ask for a cleaned copy on a new tab

Ask for a cleaned copy on its own tab plus a change list — your original stays untouched until you have checked it. If Gemini shows a small plan with an Approve button first, just approve it.

Copy-paste prompt:

Here is my imported data. Create a cleaned version as a new table in a new tab called clean data: dates in one format (YYYY-MM-DD), consistent segment names, and values as plain numbers. List every change you made.

Open Google Sheets ↗

Check it

The original stays until you checked

Cleaning must never change meaning. Three checks:

  • Compare two or three rows against the original — same account, same value, same date?
  • Read the change list: renames and reformats are fine; changed values are not.
  • Keep both tabs: the clean one is for working, the original stays as your reference.

Done

✓

Task done

You produced: A clean table you can actually analyse — every change known and checked.
