What this site is for

Purpose

A workflow, not a book

It turns the sales-cycle workshop into one place people can act from: nine steps, each with the Gem to open, an example to copy, and the output to produce.

Promise

Every step looks the same

Goal, Inputs, Open Gem, See Example, Output, Check before use. Once participants see one step, they can navigate all nine.

Audience

Sellers and leaders

The Sales Cycle is for field sales; the Leadership track lifts the same Gems to portfolio altitude.
